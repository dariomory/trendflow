# Installation

## Stable release

To install Trendflow, run this command in your terminal:

```sh
uv add trendflow-py
```

Or if you prefer to use `pip`:

```sh
pip install trendflow-py
```

The distribution on PyPI is `trendflow-py`; you still `import trendflow` in Python.

## From source

The source files for Trendflow can be downloaded from the [Github repo](https://github.com/dariomory/trendflow).

You can either clone the public repository:

```sh
git clone https://github.com/dariomory/trendflow
```

Or download the [tarball](https://github.com/dariomory/trendflow/tarball/main):

```sh
curl -OJL https://github.com/dariomory/trendflow/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd trendflow
uv sync
```
